# Full-Stack E-Shop Application

Η εφαρμογή υλοποιεί ένα πλήρες e-shop ρούχων. 

## Βασικές Λειτουργίες
1. **Διαχείριση Χρηστών:** Δημιουργία λογαριασμού (Register), Σύνδεση (Login) και Αποσύνδεση χρήστη με χρήση JWT authentication.
2. **Διαχείριση Προϊόντων:** Δημιουργία, ενημέρωση, διαγραφή (μόνο από Admin) και αναζήτηση προϊόντων.
3. **Διαχείριση Κατηγοριών:** Δημιουργία, ενημέρωση και διαγραφή κατηγοριών προϊόντων (μόνο από Admin).
4. **Καλάθι Αγορών:** Προσθήκη, αφαίρεση και αυξομείωση ποσότητας προϊόντων (αποκλειστικά για εγγεγραμμένους/συνδεδεμένους χρήστες).
5. **Παραγγελίες:** Ολοκλήρωση παραγγελίας και προβολή ιστορικού παραγγελιών.

## Τεχνολογίες που χρησιμοποιήθηκαν
* **Frontend:** React, Tailwind CSS
* **Backend:** Java Spring Boot, Spring Security (JWT)
* **Βάση Δεδομένων:** PostgreSQL (μέσω Docker container)

## Βάση Δεδομένων & Αρχικά Δεδομένα (Config)
Για την βάση δεδομένων χρησιμοποιήθηκε PostgreSQL. Στο αρχείο `Config.java` του project υπάρχει ένα αρχικό setup δεδομένων, ώστε η εφαρμογή να μπορεί να δοκιμαστεί άμεσα. Συγκεκριμένα:
* Κατά την πρώτη εκτέλεση εκτελείται η εντολή `CREATE EXTENSION IF NOT EXISTS pg_trgm;`. Αυτό είναι απαραίτητο ώστε να λειτουργήσει σωστά η αναζήτηση, καθώς χρησιμοποιήθηκαν trigrams για να είναι πιο ανεκτική σε ορθογραφικά λάθη και πιο ευέλικτη.
* Αρχικοποιείται αυτόματα ένας Admin χρήστης.
* Δημιουργούνται κάποιες αρχικές Κατηγορίες και Προϊόντα, ώστε το κατάστημα να μην είναι άδειο.

## Επιπλέον πληροφορίες

**Βάση Δεδομένων:**
1. Το username της βάσης είναι `postgres` και το password `mysecretpassword`. To port είναι το `5432` και το Database Name είναι `postgres` (όπως αναγράφεται και στο `application.properties`).
2. Για την εκτέλεση του front end χρειάζεται να εκτελεστούν οι εντολές `npm install` και έπειτα `npm start` στο terminal.
